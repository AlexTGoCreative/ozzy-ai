# MetaDefender Cluster Deployment

Integrating MetaDefender Core with MetaDefender Cluster empowers users with seamless scalability, allowing them to effortlessly handle surging volumes of file uploads and scans across distributed MetaDefender Core instances without compromising performance.

This combination delivers exceptional fault tolerance, ensuring continuous operation even if individual MetaDefender Core instances fail, while providing centralized management for streamlined oversight and policy enforcement.

More details at MetaDefender Cluster.
