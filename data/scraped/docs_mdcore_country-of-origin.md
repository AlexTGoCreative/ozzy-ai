# Country of Origin

OPSWAT Country of Origin (COO) engine enables instant detection of a file's geographic source. It analyzes digital fingerprints, metadata to identify restricted locations and vendors automatically. This allows precise filtering that blocks unauthorized data access based on origin while ensuring cross-region regulatory compliance.

**Detect and block by country**

You can configure to block or allow from specific countries for each file types

**Detect and block by vendor**

Similar to country, you can configure to block or allow from specific vendors.

**Support file types:** DLL, EXE, EXE_SFX, EFI, MSI, and CAB
