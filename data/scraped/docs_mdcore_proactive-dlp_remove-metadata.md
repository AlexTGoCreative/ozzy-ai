# Remove metadata

Most of the files have some kind of metadata which could contain sensitive information like the author, GPS coordination, etc. which can expose the organizations to risk. Proactive DLP can remove metadata contained within the file.

## Supported File Types

- Images: JPG, PNG, TIFF, GIF
- Documents: PDF, MS Office, RTF
- OpenOffice: ODT, ODS

## To set remove metadata

- Workflow Management > Workflows > "Workflow name" > Proactive DLP
