# Reputation Engine release notes

## 2.2.0

Release date: 12/1/2025

| New Features and Improvements |
|---|
**NSRL Allowlist Support:**The Reputation Engine now includes an allowlist of hashes sourced from the National Software Reference Library (NSRL). These hashes have been carefully filtered to ensure accuracy and reliability.**Third-Party Allowlist Support:**The Reputation Engine now incorporates whitelisted hashes from trusted third-party vendors. All entries are curated in-house to maintain integrity and security.
